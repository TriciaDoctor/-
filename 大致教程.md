前置条件
1. Git
2. GitHub账号
3. JetBrains IDE（IDEA、WebStorm、PyCharm等）

过程中出现任何问题请自行AI

一：首次配置（仅需一次）
1. 设置 → 版本控制 → Git → 配置Git可执行文件路径
2. 设置 → 版本控制 → GitHub → 添加账号登录

二：如何推送到GitHub
1. 点击顶部VCS → 启动版本控制 → 选择Git（选择后顶部VCS会变为Git）
2. Git → 远程管理 → 添加 → 名称：origin，URL：你的仓库地址
3. 点击提交 → 选择需要提交的文件 → 填写提交信息 → 点击提交并推送

三：如何拉取仓库内容
1. Git → 拉取 即可

四：如何克隆已有仓库
1. 文件 → 从版本控制获取 → 输入仓库URL → 选择本地目录 → 克隆

五：解决冲突
1. 拉取时若有冲突，IDE会弹出合并窗口
2. 选择保留哪边的代码（左侧本地/右侧远程/手动合并）
3. 点击应用后重新提交推送

六：查看历史记录
1. Git → 显示Git日志 → 可查看所有提交记录
